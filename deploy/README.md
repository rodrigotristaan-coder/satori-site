## Cómo publicar

**Opción 1 — Cualquier hosting estático** (Netlify, Vercel, GitHub Pages, Hostinger, etc.):
Sube TODO el contenido de la carpeta `deploy/` a la raíz del sitio. No requiere build.

**Opción 2 — Servidor propio**:
Sirve la carpeta `deploy/` como estática (Nginx, Apache, etc.). El sitio carga React/Babel desde unpkg vía CDN.
